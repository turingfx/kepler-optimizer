kepler-skills

Stream task operations toolkit for Kepler streaming engine

Kepler 是基于 JStorm 的实时流计算引擎，广泛应用于蚂蚁内部实时数据处理场景，包括实时风控、实时监控、实时特征计算等。

本工具集提供对 Kepler 流任务的完整运维能力，支持任务查询、调参优化、生命周期管理、位点重置等操作。同时兼容离线数仓 ODPS 数据源的接入配置。

Capabilities

▶ Query Task

Inspect task SQL, configuration parameters, parallelism settings, partition info, and recent operations.

⟳ Tune Task

Auto-align worker size and parallelism with upstream partitions. Configure memory, buffer, CPU slots.

⏻ Start / Stop / Restart

Control task lifecycle with graceful restart (preserve state) or immediate stop.

↺ Reset Task

Clear task state and restart from specified offset timestamp.

☰ Operation Logs

View recent operations: who did what and when.

⬡ Multi-Site Support

Switch between Kepler deployments. Auth credentials isolated per-site.

Authentication

鉴权方式

AccessKey Signature (推荐)

参数: operator + bizCode + accessKey

签名算法: digest = MD5(accessKey + timestamp)

Cookie Authentication

参数: 浏览器 Cookie 字符串

鉴权信息按站点独立存储于 ~/.kepler_auth/site_<num>.env，权限 600，敏感信息不会记录或显示。

Task Status

ACTIVE Running and processing data STOP Stopped, can be started or reset RESTARTING In transition
